# Windows 21H2to7 Language Pack Installer

This pack contains translation files for the Windows 21H2to7 Transformation Pack. Use it if you're installing the transformation pack on a non-English version of Windows.

---

## Version

**Current:** [3.6.1](https://github.com/ImSwordQueen/Win10to7LanguagePacks/tree/3.6.1+)

---

## Supported Languages

| Language | Locale Code |
|----------|-------------|
| Italian | It-IT |
| German | de-DE |
| Korean | ko-KR |
| Romanian | ro-RO |
| Russian | ru-RU |
| Polish | pl-PL |
| Turkish | tr-TR |
| Chinese Taiwan | zh-TW |

---

## How to Install

1. Download the language pack for your Windows language
2. Run the installer as Administrator
3. Restart your computer when prompted
